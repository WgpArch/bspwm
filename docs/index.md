# bspwm Dotfiles

Welcome to my custom bspwm configuration! This repository contains a highly polished, efficient, and keyboard-driven setup for the bspwm tiling window manager.

**bspwm** (Binary Space Partitioning Window Manager) is a lightweight tiling window manager for X11 that represents windows as the leaves of a full binary tree. It uses external programs for keyboard shortcuts (sxhkd) and panel (like polybar or waybar), following the Unix philosophy of doing one thing and doing it well.

## Features

- **Binary Space Partitioning**: Windows are automatically tiled in a binary tree layout
- **sxhkd Integration**: Clean, external hotkey daemon for keyboard shortcuts
- **Dual Conky Setup**: Beautiful system monitoring with sqi-conky-theme on both sides
- **Pre-configured Keybinds**: Comprehensive keyboard shortcuts for efficient workflow
- **Custom Wallpapers**: Curated selection of beautiful backgrounds
- **Lightweight & Fast**: Minimal resource usage for maximum performance
